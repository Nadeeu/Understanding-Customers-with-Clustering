# Data

The dataset used in this project is the **Online Retail II** dataset.

## Download

You can download the data from the UCI Machine Learning Repository:

🔗 [Online Retail II Dataset](https://archive.ics.uci.edu/dataset/502/online+retail+ii)

## Description

This dataset contains all the transactions occurring for a UK-based and registered, non-store online retail between 01/12/2009 and 09/12/2011. The company mainly sells unique all-occasion gift-ware.

## Instructions

1. Download the dataset from the link above
2. Extract the Excel file (`online_retail_II.xlsx`)
